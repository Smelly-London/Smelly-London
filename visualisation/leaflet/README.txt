The code for this visualisation is based on that created for the Wellcome Trust project, Women's Work: https://github.com/wellcometrust/womens-work/tree/master/visualisations/map See this repository for the original code. 

## Viewing the map.##
To run the code locally on a Unix system, download the repository then run:

python -m SimpleHTTPServer

and open http://localhost:8000/visualisation/leaflet/ in a browser to view the map.
